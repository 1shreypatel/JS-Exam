# JS-Exam

A simple **Product Management Web App** built with pure HTML, CSS & JavaScript.

### Live Demo
[Click here to see live demo](https://1shreypatel.github.io/JS-Exam/)

## Features

- ✅ Add new products (Name, Price, Category)
- ✅ Display all products in a clean list
- ✅ Filter products by Category (Electronics, Fashion, Books, Home)
- ✅ Sort products by Price (Low to High / High to Low)
- ✅ Responsive design
- ✅ Data persists using localStorage

## Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage for data persistence
